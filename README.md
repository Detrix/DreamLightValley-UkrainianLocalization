Якщо ви хочете грати Disney Dreamlight Valley українською мовою

Виконайте ці кроки:
Завантажте архів LocDB_en-US.zip із перекладом.
Перейдіть до папки з встановленою грою. Зазвичай шлях такий: D:\Steam\steamapps\common\Disney Dreamlight Valley\ddv_Data\StreamingAssets\Localization
Скопіюйте завантажений архів у цю папку з заміною
Запустіть гру. У налаштуваннях має бути обрана англійська мова (English), оскільки переклад замінює саме її.
